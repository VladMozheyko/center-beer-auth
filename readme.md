# center-beer-auth-service

Сервис аутентификации и управления пользователями, предназначенный для цифровой платформы **center.beer**.

## Описание

Данный сервис предоставляет функциональность для регистрации пользователей, аутентификации, управления токенами доступа, восстановления и изменения паролей. Он обеспечивает безопасное взаимодействие с другими компонентами системы через RESTful API.

## Основные возможности

- Регистрация и аутентификация пользователей
- Управление JWT-токенами (создание, обновление, отзыв)
- Сброс и изменение пароля
- Валидация входных данных на стороне сервера
- Безопасное взаимодействие через HTTPS

## Технологический стек

- Java 17
- Spring Boot 3.1 (модули: Web, Security, Data JPA, Validation)
- MySQL
- JWT (библиотека jjwt)
- BCrypt для хэширования паролей
- Swagger (springdoc-openapi) для документации API
- Docker для контейнеризации приложения

## Документация API

После запуска приложения, документация API доступна по адресу:
http://localhost:8080/swagger-ui.html
